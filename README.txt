\== Description ==

**See exactly which forms, buttons, and calls-to-action drive real leads—on every major analytics platform.**

LeadStream is the all-in-one event tracking and pixel injector that finally puts you in control of your marketing data. No more guessing which forms convert, which page sections win, or which pixels are firing—LeadStream lets you track, label, and optimize every conversion, everywhere.

*Supports Google Analytics (GA4), Meta Pixel, TikTok, Triple Whale, and every major form plugin (WPForms, Contact Form 7, Gravity Forms, Ninja Forms, and more).*

* **Point-and-click event tracking:** One-click starter scripts for forms, buttons, links, phone/email clicks, scrolls, and more.
* **Works with every form builder:** Instantly track WPForms, Contact Form 7, Gravity Forms, Ninja Forms, and any generic HTML form.
* **Multi-platform by design:** Send data to GA4, Facebook, TikTok, Triple Whale—just check the box. No bloat, no hard-coded scripts.
* **Fully editable:** All tracking code is visible and customizable—make it your own, or paste in any custom event logic.
* **Admin-friendly UI:** Select your samples, load with a click, and avoid rookie mistakes with real-time admin warnings.
* **No data lock-in:** Your data, your rules—no “cloud sync,” no hidden analytics fees.

Stop guessing, start tracking. Get agency-grade analytics, for free.

\== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/leadstream-analytics-injector` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'LeadStream' in your admin menu to set up your tracking and starter scripts.
4. Select your platforms and event samples, then click 'Load Starter Script' to instantly get ready-to-go tracking code for your site.
5. Paste in your own scripts or tweak the provided samples as needed.

\== Frequently Asked Questions ==

\= Will this slow down my site? =
No! LeadStream only injects the scripts and events you choose. There’s no bloat, no heavy libraries, and no cloud requests.

\= What analytics and ad platforms are supported? =
Google Analytics (GA4), Meta/Facebook Pixel, TikTok Pixel, Triple Whale, and any custom pixel or event handler you want to add.

\= Which forms are supported? =
WPForms, Contact Form 7, Gravity Forms, Ninja Forms, and any generic HTML form—plus custom events for any other plugin you need.

\= Does this collect or send my data anywhere? =
Never. All tracking is 100% on your site. No data is sent to third-party servers (except your own analytics accounts).

\= What if I already use Google Site Kit or another analytics plugin? =
LeadStream will warn you about potential double-tracking and help you avoid duplicate events.
